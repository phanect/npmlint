# npmlint

Lint your package before publishing npm packages

> [!CAUTION]
> Currently WIP. Do not use in production.

## Install

```sh
npm install -D npmlint
```

## Development

### Prerequisites

- [mise](https://mise.jdx.dev)
  - Node.js is installed by mise.

### Run local server

```sh
cd /path/to/npmlint
mise install
mise build
```
